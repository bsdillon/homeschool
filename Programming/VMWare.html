Lesson Overview

In this lesson you are going to download some free software and setup a new virtual machine (VM) on your computer. This machine will be logically separated from the one you are using so you can experiment on it without breaking your own computer.

For this VM we will use a free version of the Linux operating system. There will be some reading about Linux and then you will have lessons from time to time that will teach you more about Linux and how to use it. Linux is an operating system or OS. That means it is software that controls the machine. All the applications on the computer talk directly to the operating system. Windows and MaxOS are other OS options.

There are many flavors of Linux from Unix to Fedora and from RHEL to Ubuntu. Each version uses the same kernel. The kernel contains all the core functions of the OS while each flavor adds its own capabilities. Because they use the same kernel once you learn to Linux you will be able to Linux in any flavor. Over time you may come to like one flavor more than others.

Linux is an OS for geeks by geeks. It was built in a time before GUIs were created and it is still predominantly run from the command line. You can use the command line in other OS's, but it is less likely in Windows and far less likely on a Mac. Once you master basic Linux commands and learn to control your own computer, Windows and Mac will feel like toys. Linux really has all the power at your fingertips. That being said, with great power comes great opportunities to screw everything up. That is why we are doing all this in a VM. If you screw up, we can start all over without ruining your main computer. Good luck!

Instructions

  1. Download an Ubuntu image and save it to your machine.
  2. Download VMWare Workstation Player and install it. You will need to restart your computer.
    • If you see an error that version 15 is incompatible with your machine, try version 12.
  3. Open VMWare Workstation. Select "Create a New Virtual Machine".
  4. Select Install from "Installer disc image file". Search for and select the Ubuntu image you downloaded.
  5. Put a name on the computer that says what the OS is and why you are using it e.g. "Ubuntu-student".
  6. Select a user name and a password. You can always change this, but make it something you will remember.
    • You may see an error message saying that the host doesn't support VT. This means we have to change a setting in the computer itself.
    • Restart the computer. Interrupt the process by pushing one of the F-buttons (usually F9, F10, F11, or F12).
    • Select the option to edit system setup
    • Within the setup, look for virtualization and enable the VT.
    • Save and resume the startup. The VM should start now.
  7. Once the image is created and the machine runs, log in using that user name and password.
  8. Note that you may have to install other software on VMWare or it cannot properly configure your machine.
